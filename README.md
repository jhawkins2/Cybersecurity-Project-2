# Cybersecurity-Project-2
Offensive and Defensive Security Project

This Presentation explains the following:
1. Vulnerabilities exploited of a Capstone VM
2. Analysis of data by viewing logs using Kibana
3. Proposed mitigation strategies

Throughout this Offensive and Defensive project, I played the role of a hacker to infiltrate a machine. Also, I played the role as a defensive security rep by viewing logs and discovering the most important and best ways to protect the system
